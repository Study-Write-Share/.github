## 📦 核心项目

### 🚀 [Study_Startup_Diary](https://github.com/Study-Write-Share/Study_Startup_Diary)

> **Ubuntu & ROS 学习笔记与环境配置**

包含 Ubuntu 与 ROS 相关的学习笔记、环境配置和示例代码。记录从环境搭建到案例运行的完整步骤与问题解决过程，适合初学者进行环境复现与进阶练习。

**关键特性：**
- ✅ 详细的环境搭建步骤

## 📖 如何使用本组织的资源

### 快速开始

1. **📂 浏览项目**
   - 访问感兴趣的项目仓库
   - 阅读 `README.md` 获取快速开始指南

2. **🔍 查找资源**
   - 在仓库中按目录查找配置脚本、示例代码与笔记
   - 多数项目会在 README 中指向关键文件

3. **💬 寻求帮助**
   - 先查看 Issues 与 Discussions，看是否已有解决方案
   - 使用合适的模板新建 Issue 提出问题
   - 在 Discussions 中进行技术讨论

### 推荐学习路径

```
1️⃣ HowTo (配置基础工具) 
    ↓
2️⃣ Study_Startup_Diary (学习 ROS/Ubuntu)
    ↓
3️⃣ 参与贡献 (分享你的经验)
```

### 贡献流程

```bash
# 1. Clone 仓库（组织成员可直接 clone）
git clone https://github.com/Study-Write-Share/<repository-name>.git

# 2. 创建新分支
git checkout -b feature/your-feature-name

# 3. 进行修改并提交
git add .
git commit -m "feat: 描述你的改动"

# 4. 推送到远程
git push origin feature/your-feature-name

# 5. 创建 Pull Request
# 选择合适的 PR 模板，填写详细信息
```
